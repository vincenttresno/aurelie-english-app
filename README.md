# Aurelie English Learning App

Adaptive Englisch-Lernplattform mit Claude AI.

## Features

- Aktive Produktion (tippen statt Multiple-Choice)
- Dynamische Übungsvariationen
- Spaced Repetition System
- Fehleranalyse mit gezielten Erklärungen
- Kontextbezogenes Feedback (Past Simple vs Present Perfect)

## Setup

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY="your-key-here"
streamlit run app.py
```
